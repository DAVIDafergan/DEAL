# נוסחי הודעות — Dealim

> **⚠️ נוסחים אלה מחייבים אישור עורך דין לפני הפעלה בייצור.**
> הם נכתבו לצורך בניית המנגנון הטכני בלבד (Step 5 — הגשר לבעלי הצימרים). שליחת הודעות אמיתית
> מנוטרלת כברירת מחדל מאחורי `PROPERTY_MESSAGING_ENABLED=false`
> (ראו `server/services/complianceMessaging.js`) — אין להפעיל בלי אישור.
>
> כל הודעה יוצאת חייבת לכלול, בהתאם לכללי ההודעות המחייבים: זיהוי מלא של השולח (שם הפלטפורמה
> + קישור), פרטי הבקשה בלבד (לא הודעות שיווקיות יזומות), ושורת הסרה. תשובת "הסר" מפעילה הסרה
> אוטומטית ללא צורך באישור אדמין. מקסימום הודעה אחת לבקשת הזמנה + תזכורת אחת לכל היותר.

מקור קוד: `server/services/complianceMessaging.js` בונה את הטקסט בפועל מהתבניות שלהלן (עם
placeholders מוחלפים) — שינוי נוסח כאן דורש עדכון תואם שם.

---

## 1. הודעת בקשת הזמנה ראשונה (לנכס `unclaimed`)

נשלחת **פעם אחת בלבד** לכל בקשת הזמנה, מיד עם קבלתה, לטלפון/וואטסאפ הרשום על הנכס.

```
שלום, קיבלתם בקשת הזמנה חדשה דרך Dealim (dealim.org) עבור הנכס "{property_name}" שלכם.

תאריכים: {check_in} – {check_out}
מספר אורחים: {guest_count}
שם הפונה: {customer_name}
טלפון: {customer_phone}
{message_line}

הנכס שלכם מוצג כרגע ב-Dealim ממקורות פומביים. לניהול מלא של הנכס (עדכון פרטים, תמונות,
זמינות) — היכנסו לקישור: {property_url}

להסרת הנכס ולהפסקת פניות — השב הסר
```

## 2. תזכורת (מקסימום אחת, אם הבקשה עדיין ללא מענה)

נשלחת **פעם אחת בלבד**, לא לפני יומיים מההודעה הראשונה, ורק אם לא נשלחה עדיין תזכורת לבקשה
זו (`reminder_sent_at IS NULL`) והבקשה עדיין `pending`.

```
תזכורת: יש לכם בקשת הזמנה ממתינה ל-{property_name} דרך Dealim, מ-{customer_name}
(תאריכים: {check_in} – {check_out}). הבקשה מ-{days_ago} ימים.

לצפייה בפרטים המלאים: {property_url}

להסרת הנכס ולהפסקת פניות — השב הסר
```

## 3. אישור הסרה (תגובה אוטומטית ל"הסר")

נשלחת מיד לאחר עיבוד תשובת "הסר" (בין אם דרך תגובת וואטסאפ ובין אם דרך עמוד `/remove`).

```
הנכס שלכם הוסר מ-Dealim ולא יופיע יותר באתר. לא תישלחנה אליכם הודעות נוספות ממספר זה.

אם ההסרה בוצעה בטעות, ניתן לפנות אלינו: {support_email}
```

## 4. אישור הסרה — עמוד /remove (מסך הצלחה, לא הודעה)

```
הנכס הוסר בהצלחה מ-Dealim ונחסם לצמיתות. המספר/הדומיין נוספו לרשימת ההסרה שלנו ולא ייסרקו
או ייווצרו מחדש בעתיד.
```
